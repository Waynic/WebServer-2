WebServer
=
基本功能
-
* 可配置IP地址、监听端口和主目录
* 支持服务的启动和关闭
* 对每个请求能够创建单独的响应线程
* 支持多种类型文件的输出：

		图片：jpe, jpeg, jpg, gif, png
		文本：txt, stm, htm, html, css, doc, docx
		音频：mp3
		其他：pdf
